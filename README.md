Формат: продуктовый (startup-style)

Vision продукта

Courier Finance — это Telegram Mini App для курьеров, предназначенное для быстрого и точного расчёта заработка за смену, ведения финансовой статистики и анализа реальной прибыльности работы.
Приложение показывает доход:

с налогом,

без налога,

чистую прибыль с учётом бензина,

а также позволяет:

анализировать эффективность работы,

отслеживать динамику доходов,

сравнивать себя с другими курьерами через рейтинги.

Основная ценность:
«Курьер в любой момент знает, сколько он реально зарабатывает и насколько выгодна его работа».

Целевая аудитория

Основная:

курьеры служб доставки,

фриланс-доставщики,

частные курьеры.

В перспективе:

менеджеры смен,

логистические компании,

агрегаторы курьеров.

Авторизация и профиль

Авторизация:

автоматическая через Telegram WebApp API,

используется Telegram user.id как уникальный идентификатор,

без логинов и паролей.

Профиль пользователя:

ставка за минуту (по умолчанию: 0,54),

цены зон:

зона 1 → 196,

зона 2 → 212,

зона 3 → 239,

налоговый коэффициент (например 0.87, редактируемый),

валюта,

цель по доходу,

включение/выключение учёта бензина,

участие в рейтинге.

Все базовые тарифы задаются с налогом.

Основной User Flow

Пользователь открывает Mini App

Происходит автоматическая авторизация

Открывается экран расчёта смены

Пользователь вводит:

дату (по умолчанию сегодня),

количество отработанных минут,

количество заказов:

зона 1,

зона 2,

зона 3,

километраж,

сумму, потраченную на бензин,

Нажимает «Рассчитать»

Приложение показывает:

доход с налогом,

доход без налога,

чистую прибыль (с учётом бензина),

Нажимает «Сохранить в статистику».

Расчётная логика

Доход за время:

timeIncome = minutes × ratePerMinute


Доход за заказы:

ordersIncome =
(zone1 × priceZone1) +
(zone2 × priceZone2) +
(zone3 × priceZone3)


Общий доход с налогом:

totalWithTax = timeIncome + ordersIncome


Доход без налога:

totalWithoutTax = totalWithTax × taxCoefficient


Чистая прибыль:

netProfit = totalWithoutTax − fuelCost


В статистике всегда отображаются:

прибыль без учёта бензина,

прибыль с учётом бензина.

Структура данных одной смены

{
  "date": "2026-01-17",

  "zone3": 2,
  "kilometers": 82,
  "fuelCost": 1000,
  "totalWithTax": 3500,
  "totalWithoutTax": 3045,
  "netProfit": 2045
}
